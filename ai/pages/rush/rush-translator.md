## Consigne

Nous souhaitons un CLI qui peut être utilisée de la manière suivante:

```bash
./translator [sentence]
```

La sortie est la proposition de traduction de la phrase donnée en paramètre.

## Exemple:

```bash
./translator "bonjour comment va tu"
```
 Output:

```bash
hello how are you
```

## Consigne

Nous souhaitons une IA permettant de reconnaitre un objet ou alors un visage à partir
d'un flux vidéo live.

```bash
./reconizer
```

La sortie doit être le même flux vidéo mais en plus, un rectangle autour de l'objet ou
du visage reconnu.

## Exemple:

```bash
./reconizer
```
